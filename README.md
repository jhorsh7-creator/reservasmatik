Mi primer repositorio con GITHUB
**ReservasMatik** es un proyecto desarrollado con la finalidad de poner en práctica el uso de **Git y GitHub** para el trabajo colaborativo. El repositorio permite gestionar y documentar el proceso de creación, versionamiento y control de cambios de un proyecto, aplicando buenas prácticas de desarrollo colaborativo.
